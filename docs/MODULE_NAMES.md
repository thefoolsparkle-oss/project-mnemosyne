# Module Names

为了更容易记，系统模块以中文名为主，英文名只作为代码内部代号。

```text
人格生成器     Forge       根据用户选择和描述生成初始人格
记忆档案员     Archivist   提取、编号、分层、链接长期记忆
用户画像器     Mirror      整理用户互动偏好、情绪模式、被回应方式
人格调整器     Sculptor    根据记忆和反馈提出人格修订建议
形象生成器     Visage      管理外貌设定、头像 prompt、图片生成
主聊天 AI      Core        读取整理后的上下文并生成回复
```
