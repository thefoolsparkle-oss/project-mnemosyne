隐私与数据说明

这是当前本地开发版的数据处理说明，用于帮助测试者理解数据会怎样保存、导出和清理。

保存哪些数据

系统会在本地数据库保存账号、资料、人格、聊天记录、群聊、记忆、表达偏好、主动联系候选反馈和管理员审查记录。上传头像和表达素材会保存在本地上传目录。

模型调用

真实聊天可能会把必要的对话上下文、人格资料和记忆片段发送给配置的模型服务。验证脚本和管理台健康概览只读取本地日志，不会为了检查随意调用真实模型。

游客模式

游客账号默认三天后到期。到期后，游客账号及其关联人格、聊天、记忆和上传文件会被清理；清理批次会留下本地审计日志。转成正式账号后会保留当前游客数据。

导出与删除

资料页可以导出账号级 JSON 数据，人格资料页和已删除面板也支持人格级导出。人格彻底清除前会要求输入名字确认；清除后对应人格资料与单聊数据不可恢复。

本地密钥

API Key 应只放在本机环境变量或本地 .env 文件中，不应写入仓库。账号导出和本地备份脚本不会复制 .env 文件或 session token。
